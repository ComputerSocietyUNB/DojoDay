Dojo Day 2017
=============

Repositório para formular o material que o Coding Dojo UnB irá apresentar no `Dojo Day 2017`_.

.. _Dojo Day 2017: https://www.sympla.com.br/dojo-day-3__144322

Tools
-----

* Virtualenv
* Python 3.x
* Unittest
* Flask
* Selenium
* HTML

Create
------

* Criar ambiente env/
* Ativar o ambiente env/
* Executar o script:

    $ python3 setup.py install

Run
---

* Entrar em src/
* Executar o script:

    $ python3 tests.py
